Pipeline lets you take control over streams with a plumbing tool-kit.

API Reference

Pipe

The most basic structure of a pipeline, Pipe takes a in function that receives the next connection.

        
          new Pipe()
            .in(function(evt,next){
              if (evt.type == 'mousedown'){ this.out.open() }
              if (evt.type == 'mouseup'){ this.out.close() }
              next(evt);
          })

Tee

The most basic structure of a pipeline, Pipe takes a in function that receives the next connection.

        

          new Tee()
          .connectTo(function(req, res){
            return (req.url.indexOf('blue') !== -1);
          },new Pipe()
          .in(function(req, res){
            res.end(req.url);
          }))

          .connectTo(function(req, res){
            return (req.url.indexOf('red') !== -1);
          },new Pipe()
          .in(function(req, res){
            res.end(req.url);
          }))
